## How to use

**Option 1 — Open directly in your browser**  
Download any `.html` file and open it in Chrome, Firefox, or Safari.  
Your data is saved automatically in the browser.

**Option 2 — Add to your phone home screen (recommended)**  
1. Open the file in your mobile browser  
2. Tap the Share button (iOS) or Menu → Add to Home Screen (Android)  
3. It installs like an app with the custom icon  

**Option 3 — Host on GitHub Pages**  
1. Fork or upload this repository to GitHub  
2. Go to Settings → Pages → set source to `main` branch  
3. Your tracker is live at `https://yourusername.github.io/repo-name/`

## Backup your data

Your tasks are stored in your browser's local storage.  
Before switching devices or clearing your browser cache:  
→ Open the **Progress** tab → tap **Save backup**
